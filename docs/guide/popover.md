# Popover mode

```vue
<date-picker auto-submit popover />
```
<ClientOnly>
  <date-picker auto-submit popover />
</ClientOnly>


```vue
<date-picker auto-submit popover="bottom-left" />
```
<ClientOnly>
  <date-picker auto-submit popover="bottom-left" />
</ClientOnly>


accepted:

  `true` | `false`
  
  `top-left` | `top-right`
  
  `bottom-right` | `bottom-left`
  
  `left-top` | `left-bottom`
  
  `right-top` | `right-bottom`
