# Popover mode

```vue
<date-picker auto-submit popover="right" />
```
<ClientOnly>
  <date-picker auto-submit popover="right" />
</ClientOnly>


```vue
<date-picker auto-submit popover="bottom-left" />
```
<ClientOnly>
  <date-picker auto-submit popover="bottom-left" />
</ClientOnly>


accepted:

  `true` | `false`
  
  `top-left` | `top-right`
  
  `bottom-right` | `bottom-left`
  
  `top` | `bottom`
  
  `right` | `left`
